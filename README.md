# Mobile Usage Behavior Analysis - India

## Project Overview
This project analyzes mobile usage behavior in India using Excel.  
The goal is to identify patterns in mobile usage, screen time, app categories, and user demographics.

## Tools Used
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Charts & Visualizations

## Key Analysis
- Average daily screen time analysis
- App category usage trends
- Age group mobile usage patterns
- User behavior insights
